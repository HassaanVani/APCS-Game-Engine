================================================================================
POLYMORPHISM IN ACTION:
================================================================================

The game engine treats ALL levels the same way:

  // In MainGUI.java or Main.java:
  engine.addLevel(new ForestLevel(...));      // Student 1
  engine.addLevel(new CaveLevel(...));        // Student 2
  engine.addLevel(new YourLevel(...));        // Student 3
  
  // Engine runs them all polymorphically:
  for (Level level : levels) {
      level.start(player);  // Different implementation each time!
  }

This is THE core teaching point - polymorphism enables collaboration!
